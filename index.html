A powerful Go framework for rapid API development with clean architecture patterns

🚀 Fast Development

CLI tool for rapid scaffolding and code generation. Build APIs in minutes, not hours.

🏗️ Clean Architecture

Repository, Service, Handler pattern with dependency injection for maintainable code.

🔧 Code Generation

Generate models, handlers, services, repositories, and routes with a single command.

🌐 High Performance

Built on top of Go Fiber framework for lightning-fast HTTP performance.

🗄️ GORM Integration

Advanced ORM with scoping support and database management.

🎯 Simplified Handlers

Generate lightweight handlers with text responses for rapid prototyping.
